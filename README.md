# XENØr Site

Public-facing site for XENØr.

This repository contains the public website for the XENØr ecosystem, including protocol context, research direction, and the project's external surface.

## Purpose

The site serves as the main public entry point for XENØr and is intended to communicate:

- protocol overview
- research direction
- ecosystem context
- public links and references

## Related Repositories

- **xenor-core** — core protocol and systems work
- **xenor-sim** — simulation and modeling environment
- **xenor-sale** — archived early sale experiment, kept for reference only
